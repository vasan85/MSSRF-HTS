Auto-tracked on every record
  • 📅 Created At — timestamp of record creation

Data Validation Rules
  • 🏘️ Village selection is mandatory
  • 📱 Mobile number format validated (10 digit)
  • 🔒 Household ID is non-editable after creation
  • 🚫 Duplicate project enrollment on same date blocked
  • 📆 Enrollment date cannot exceed project end date
  • ✅ Project must be active for linking
